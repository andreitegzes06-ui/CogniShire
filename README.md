# Cognishire: A Map of Human-Computer Interaction

An Obsidian vault for studying Human-Computer Interaction.

## How to run

1. Download or clone this repository.

```bash
git clone https://github.com/andreitegzes06-ui/map_of_HCI.git
```

2. Open **Obsidian**.

3. Click **Open folder as vault**.

4. Select the cloned project folder:

```text
map_of_HCI
```

5. Start from:

```text
Human-Computer Interaction.md
```

## Recommended setup

For the intended visual style:

1. Open **Settings** in Obsidian.
2. Go to **Appearance**.
3. Enable the included CSS snippets.
4. Open the vault in **Reading View**.

Some styling and diagrams may look different in GitHub preview. Use Obsidian for the best experience.
